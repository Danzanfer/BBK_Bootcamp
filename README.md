# CheapCoffee

## Descripción
Esta aplicación web está diseñada para ayudar a los usuarios a encontrar los cafés más económicos y cercanos según su ubicación actual y sus preferencias de tiempo de caminata.

El proyecto busca solucionar la necesidad de encontrar opciones rápidas y asequibles de café sin tener que caminar largas distancias o gastar de más, enfocado principalmente en estudiantes y trabajadores jóvenes, gente en paro y turistas.

## Estructura del Prototipo
El prototipo consta de 3 vistas principales conectadas entre sí para simular el flujo de la aplicación:

* **index.html:** Página principal donde el usuario selecciona la distancia máxima y el tiempo de caminata deseado mediante controles deslizantes[cite: 53].
* **results.html:** Muestra la mejor opción de café encontrada, un mapa con la ruta y una tabla con alternativas cercanas. También permite dirigir al usuario hacia un formulario estático para añadir un nuevo lugar a la base de datos.
* **new_place.html:** Formulario estático para simular la capacidad de añadir un nuevo lugar a la base de datos.
